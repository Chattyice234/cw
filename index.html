ChattyWebsite Announcements

For OGSD, both ChattyWebsite and Barney's Website have been getting blocked. To prevent this, please press Ctrl + S, and in the popup, where it says "Webpage, single file", click on it, and then press "Webpage, HTML Only". Then, name your file to your liking, and then press Save. This way, you can download ChattyWebsite as a file to bypass restrictions. PLEASE make sure the file is not downloaded as .mhtml, it does not run JS which is required to make the site function. The HTML filetype does allow JS to run. If your device storage is low, (recommened even if it is not), move the file into a permanent folder like "My Files" or "Recovered Files from Google Drive". Either one works, and other folders that are permanent work. Or, fork this project on GitHub: https://github.com/Chattyice234/cw
